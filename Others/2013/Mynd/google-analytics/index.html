• Google Analytics
  1. Make sure that you have "E-commerce tracking" turned on in your Google Analytics account.
    To do this, follow these steps:
    a. Log in to analytics and click your property id/site name
    b. Click the orange 'Admin' button in the top right of the page
    c. Select your profile
    d. Select 'Profile Settings'
    e. Make sure 'Yes, an E-commerce Site' is selected
    f. Also, if you want to track site search make sure 'Do Track Site Search' is selected. In the 'Query Parameter' text box, enter filter_name and also make sure "Strip query parameters out of URL " is NOT checked. (You might need to edit this if SEO URLs are NOT turned on in OpenCart)

    It should now look like the image below. Click Apply to save the settings:

  2. Now click the 'Goals' tab then click CREATE A GOAL.
  3. Select the 'Place an order' option under REVENUE and click the 'Next step' button.
  4. Make sure the Type is set to 'Destination' and click the 'Next step' button.
  5. You then need to copy the settings on the image below (scroll down below the image to copy and paste the settings & go to step 18 to copy and paste the funnel steps you see here):
    Note: If using an extension which changes checkout functionality, your goal funnel steps will be different to the image - scroll down to step 18 to see the steps for popular checkout extensions)



    For the goal destination, select Regular expression and enter this:
    /index\.php\?route=*checkout/success*

    Make sure Value is OFF & Turn Funnel ON.
  6. Setting up the goal funnel:
    Set Required to NO.

    Select which type of checkout you are using, then copy and paste the info into the relevant fields:

    Default/Standard Checkout

  7. Save Goal to complete.
  8. That is it! Your site is now all set up with complete analytics tracking.
